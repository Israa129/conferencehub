# ConferenceHub

ConferenceHub is a full-stack web application developed as an academic team project at the Université de Technologie de Belfort-Montbéliard (UTBM).

## Technologies

- Angular
- Laravel
- PHP
- MySQL
- MongoDB
- REST API
- Git / GitHub

## Main Features

- User authentication
- Conference management
- Speaker management
- Participant registration
- QR Code generation
- Certificates
- Admin Dashboard
- Responsive user interface
